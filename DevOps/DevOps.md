# DevOps

**DevOps**是一种**思想/方法论**，它涵盖开发、测试、运维的整个过程
- 强调软件开发人员与软件测试、软件运维、质量保障（QA）部门之间有效的沟通与协作
- 强调通过自动化的方法管理软件变更、软件集成

使软件从构建到测试、发布更加快捷、可靠，最终按时交付软件
